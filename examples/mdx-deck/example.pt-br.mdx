# React web

## Novidades

- Componente `Fab` totalmente reescrito;
- Remoção de [vulnerabilidade do lodash](https://snyk.io/vuln/SNYK-JS-LODASH-590103);
- Remoção de erros de tipo dos componentes `Fab`, `Tooltip` e `Typography`;
- Atualizações gerais de segurança e desempenho;

---

## Novidades para quem contribui

- Suporte oficial ao novo TypeScript 4;
- Atualizações de dependências de desenvolvimento, incluindo contribuição para o `commitizen`;
- _Linting_ mais rápido para o seu _code style_ não ficar mal;
- Remoção temporária da dependência `alex` (vulnerabilidades encontradas).

Confira o [diff no GitHub](https://github.com/natura-cosmeticos/natds-js/compare/v0.17.11...v0.18.20)
