## Installation

```sh
// with npm
npm install --save @naturacosmeticos/natds-rn react-native-vector-icons
react-native link react-native-vector-icons
// with yarn
yarn add @naturacosmeticos/natds-rn react-native-vector-icons
react-native link react-native-vector-icons
```
