## FAB Props

| Name     |                Type                 |  Default  |                                                                   Description                                                                   |
| :------- | :---------------------------------: | :-------: | :---------------------------------------------------------------------------------------------------------------------------------------------: |
| variant  |        'round' \| 'extended'        |  'round'  |                                                          The component variant to use                                                           |
| color    | 'primary' \| 'secondary' \| 'light' | 'primary' |                                     The color to be used from the theme. 'light' uses the 'paper' namespace                                     |
| size     |   'large' \| 'medium' \| 'small'    |  'large'  |                                                            The size of the component                                                            |
| disabled |               boolean               |   false   |                                        Disables the button on the component and applies specific styling                                        |
| children |           React.ReactNode           |           | The children of the component. For 'round' variant, we recommend not using too many letters, as it can cause distortions on the component shape |
| classes  |               string                |           |                                                       CSS classes to use on the component                                                       |
| style    |         React.CSSProperties         |           |                                                    Style to be used inline on the component                                                     |
