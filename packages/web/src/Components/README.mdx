# Component folder structure

- `I{Component}Props.ts` (optional): interface with properties definition;
- `index.story.tsx`: the Playground (Interactive) story;
- `index.tsx`: exports main component (and related ones);
- `sections.tsx`: variations for stories;
- `README.mdx`: documentation and non-interactive stories;
