# Component anatomy

## Folder structure

### Component

- `I{Component}Props.ts` (optional): interface with properties definition;
- `index.tsx`: component production code;
- `index.test.tsx`: component test code;

### Stories & docs

- `README.mdx`: documentation and non-interactive stories;
- `index.story.tsx`: the interactive story (for Playground);
- `stories.tsx`: all variations for stories;
- `stories.test.tsx`: tests for all variations;
