## API

```jsx highlight-line="2"
import { Badge } from '@naturacosmeticos/natds-web';

<Badge color="primary">
  Primary
</Badge>;
```

---
---

| Name                      | Type                                            | Default                               | Description
| :--                       | :-:                                             | :-:                                   | :-:
| id                        | string                                          |                                       | The input id property
| theme                     | IThemeWeb                                       | Natura Light Theme                    | The theme provided for the component rendering
| color                     | 'primary' \| 'secondary' \| 'info' \| 'error' \| 'warning' \| 'success' \| 'light' \| 'dark' | 'primary' | The color to use from the theme
| type                      | 'standard'                                      | 'standard'                            | The type of the badge
| classes                   | string                                          |                                       | The classes to use on the container 'div'
| style                     | React.CSSProperties                             |                                       | Inline style for the container 'div'
| children                  | React.ReactNode                                 |                                       | The children that will be placed inside the badge
