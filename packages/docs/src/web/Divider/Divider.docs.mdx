## API

```jsx highlight-line="2"
import { Divider } from '@naturacosmeticos/natds-web';

<Divider variants="fullWidth" />;
```

---

---

| Name      |              Type              |   Default   |                                        Description                                         |
| :-------- | :----------------------------: | :---------: | :----------------------------------------------------------------------------------------: |
| variant   | 'fullWidth', 'inset', 'middle' | 'fullWidth' |                                    The variant to use.                                     |
| component |          elementType           |    'hr'     | The component used for the root node. Either a string to use a DOM element or a component. |
| absolute  |              bool              |    false    |                              Absolutely position the element.                              |
| classes   |             object             |             |                  Override or extend the styles applied to the component.                   |
| light     |              bool              |    false    |                      If true, the divider will have a lighter color.                       |

For more details consult the full [Material Avatar API](https://material-ui.com/api/divider/)
