## API

```js highlight-line="2"
import { TextField } from '@naturacosmeticos/natds-web';

<TextField
  label="Label"
  helpText="Help message"
  placeholder="Placeholder"
  status="success"
  disabled
  required
/>;
```

| Name                      | Type                                            | Default                       | Description
| :--                       | :-:                                             | :-:                           | :-:
| id                        | string                                          |                               | ID is applied to the Input element and is used in the label "htmlFor".
| label                     | string                                          |                               | Label is the text used above the Input element. It describes the name of the field.
| helpText                  | string                                          |                               | HelpText is the text used below the Input element.
| placeholder               | string                                          |                               | Text to display when the input is empty.
| required                  | boolean                                         | false                         | Whether the field is required or not.
| disabled                  | boolean                                         | false                         | Disables the input for user interactions.
| status                    | string                                          | 'success', 'error', undefined | Sets whether the input is in error or success state.
