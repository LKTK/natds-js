## API

```jsx highlight-line="2"
import { Checkbox, Radio, Switch } from '@naturacosmeticos/natds-web';

//Enabled
<Checkbox color="primary" />;
<Radio color="primary" />;
<Switch color="primary" />;

//Disabled
<Checkbox disabled />;
<Radio disabled />;
<Switch disabled />;
```
