## API

```jsx highlight-line="2"
import { Divider } from '@naturacosmeticos/natds-web';

<Divider variants="fullWidth" />;

// To apply subheader, e.g with grid
<GridContainer
  spacing={3}
  style={{
    width: 360,
    height: 140,
    margin: 0,
    border: 1,
    borderStyle: 'ridge',
  }}
>
  <GridItem md={12} style={{ padding: '32px 0 0' }}>
    <Divider variant="fullWidth" />
    <div style={{ margin: '8px 15.5px' }}>
      <Typography variant="caption" color="textSecondary">
        Subheader with fullWidth
      </Typography>
    </div>
  </GridItem>
  <GridItem md={12} style={{ padding: '32px 0 0' }}>
    <Divider variant="inset" />
    <div style={{ margin: '8px 72px' }}>
      <Typography variant="caption" color="textSecondary">
        Subheader with inset
      </Typography>
    </div>
  </GridItem>
</GridContainer>;
```

For more advanced usages, check [Material UI Divider docs](https://material-ui.com/components/dividers/)

## Divider Props

| Name        |              Type              |   Default    |                                                             Description                                                             |
| :---------- | :----------------------------: | :----------: | :---------------------------------------------------------------------------------------------------------------------------------: |
| absolute    |              bool              |    false     |                                                  Absolutely position the element.                                                   |
| classes     |             object             |              | Override or extend the styles applied to the component. See [CSS API](https://material-ui.com/pt/api/switch/#css) for more details. |
| component   |          elementType           |     'hr'     |                     The component used for the root node. Either a string to use a DOM element or a component.                      |
| light       |              bool              |    false     |                                           If true, the divider will have a lighter color.                                           |
| orientation |    'horizontal', 'vertical'    | 'horizontal' |                                                      The divider orientation.                                                       |
| variant     | 'fullWidth', 'inset', 'middle' | 'fullWidth'  |                                                         The variant to use.                                                         |
