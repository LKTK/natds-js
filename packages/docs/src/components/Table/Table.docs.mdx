## API

```
import { 
  TableContainer,
  Table,
  TableHead,
  TableRow,
  TableCell,
  TableBody
} from '@naturacosmeticos/natds-web';

  const align = 'center'

  return (
    <Table
        dividers={true}
        striped={true}
        rounded={true}
      >
        <TableHead>
          <TableRow>
            <TableCell align={align}>Header Cell</TableCell>
            <TableCell align={align}>Header Cell</TableCell>
            <TableCell align={align}>Header Cell</TableCell>
          </TableRow>
        </TableHead>
        <TableBody>
          <TableRow>
            <TableCell align={align}>Cell</TableCell>
            <TableCell align={align}>Cell</TableCell>
            <TableCell align={align}>Cell</TableCell>
          </TableRow>
          <TableRow>
            <TableCell align={align}>Cell</TableCell>
            <TableCell align={align}>Cell</TableCell>
            <TableCell align={align}>Cell</TableCell>
          </TableRow>
          <TableRow>
            <TableCell align={align}>Cell</TableCell>
            <TableCell align={align}>Cell</TableCell>
            <TableCell align={align}>Cell</TableCell>
          </TableRow>
          <TableRow>
            <TableCell align={align}>Cell</TableCell>
            <TableCell align={align}>Cell</TableCell>
            <TableCell align={align}>Cell</TableCell>
          </TableRow>
        </TableBody>
      </Table>
  );
};
```

For more advanced usages, check [Material UI Table docs](https://material-ui.com/pt/components/table/).

## Table Props

| Name                  | Type                                  | Default         | Description                                                                                                                         |
| --------------------- | ------------------------------------- | --------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| children              | node                                  |                 | The content of the table, normally TableHead and TableBody.                                                                         |
| classes               | object                                |                 | Override or extend the styles applied to the component. See CSS API below for more details.                                         |
| component             | elementType                           | 'table'         | The component used for the root node. Either a string to use a DOM element or a component.                                          |
| padding               | 'default'| 'checkbox'| 'none'         | 'default'       | Allows TableCells to inherit padding of the Table.                                                                                  |
| size                  | 'small'| 'medium'                     | 'medium'        | Allows TableCells to inherit size of the Table.                                                                                     |
| stickyHeader          | bool                                  | false           | Set the header sticky.⚠️ It doesn't work with IE 11.                                                                               |
| rounded               | bool                                  | true            | Set border rounded on Table                                                                                                         |
| striped               | bool                                  | true            | Set stripes on Table                                                                                                                |
| dividers              | bool                                  | true            | Set dividers on Table                                                                                                               |
