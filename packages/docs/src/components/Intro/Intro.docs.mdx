## API

```jsx highlight-line="2"
import { Intro } from '@naturacosmeticos/natds-web';

<Intro 
  title="Heading"
  titleVariant="subtitle1"
  detail="Lorem ipsum dolor sit amet, consectetur adipiscing elit."
  detailVariant="body1"
/>
```

## Props

| Name              | Type                                                                                                                                                 |  Default                            | Description                                                                                                                                     |
| :---------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------: | :---------------------------------: | :---------------------------------------------------------------------------------------------------------------------------------------------: |
| title             | element type                                                                                                                                         |                                     | Node or text to apply to the Title                                                                                                              |
| detail            | element type                                                                                                                                         |                                     | Node or text to apply to the Detail                                                                                                             |
| variantTitle      | 'h1' \| 'h2' \| 'h3' \| 'h4' \| 'h5' \| 'h6' \| 'subtitle1' \| 'subtitle2' \| 'body1' \| 'body2' \| 'caption' \| 'button' \| 'overline' \| undefined | 'subtitle1'                         | Defines the font size, line height, font family and font weight based on the provided theme for the Title                                       |
| variantDetail     | 'h1' \| 'h2' \| 'h3' \| 'h4' \| 'h5' \| 'h6' \| 'subtitle1' \| 'subtitle2' \| 'body1' \| 'body2' \| 'caption' \| 'button' \| 'overline' \| undefined | 'body1'                             | Defines the font size, line height, font family and font weight based on the provided theme for the Detail                                      |
| componentTitle    | element type                                                                                                                                         | 'h3'                                | The html element used for the Title                                                                                                             |
| componentDetail   | element type                                                                                                                                         | 'h3'                                | The html element used for the Detail                                                                                                            |
