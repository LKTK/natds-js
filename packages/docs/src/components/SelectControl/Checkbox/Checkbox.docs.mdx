## API

```jsx highlight-line="2"
import { Checkbox } from '@naturacosmeticos/natds-web';

//Enabled
<Checkbox color="primary" />;

//Disabled
<Checkbox disabled />;
```

For more advanced usages, check [Material UI Checkbox docs](https://material-ui.com/components/checkboxes/)
